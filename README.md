## Mesto Russia

Сайт, позволяющий просматривать и добавлять карточки с фотографиями, ставить им лайки, а также редактировать профиль.


#### _Технологии, используемые в проекте:_

*   HTML5
*   CSS3
*   JavaScript
*   Методология БЭМ
*   NPM
*   WebPack
